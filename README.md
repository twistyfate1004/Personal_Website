# Personal Website

This is a purely vibe-coded personal website project. The early-stage development was mainly carried out in Claude Code using GLM-4.7. In the later stage, the project was handed over to Codex with GPT-5.5 for some final polishing work.

The core purpose of this project was to test the coding capabilities of these two coding agents and the two models behind them. All prompts used were just simple one-sentence instructions, and the project was never developed as a serious production project. Therefore, this is not an ideal example of a vibe-coding project.

Tip: Except for this sentence, everything else was generated by AI.
